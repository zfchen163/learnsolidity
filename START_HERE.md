## 🛠️ 快速环境检查

运行这些命令检查你的环境：

```bash
# 检查 Go
go version
# 应该显示：go version go1.21 或更高

# 检查 Node.js
node --version
# 应该显示：v18.0.0 或更高

# 如果没有安装，运行：
# macOS:
brew install go node
```

## 💡 学习建议

### ✅ 推荐的学习方式

- **每天 1-2 小时**，持续 2 周
- **按顺序学习**，不要跳课
- **动手实践**，运行每一个例子
- **改改代码**，看看会发生什么

### ❌ 不推荐的方式

- 一次性学完（会消化不良）
- 只看不做（学不会）
- 死记硬背（忘得快）
- 遇到问题就放弃（坚持就是胜利）
