# Solidity 全栈开发实战课程 🚀

欢迎来到 **Learn Solidity**！这是一套专门为初学者设计的 Solidity 和 Web3 开发实战教程。无论你是完全的编程新手，还是有经验的 Web2 开发者，这里都有适合你的内容。

## 📚 课程结构

本课程分为三个阶段，从基础概念到高级应用，带你一步步掌握区块链开发技能。

### 第一阶段：基础入门 (Basics)
- **[第01课：区块链基础](lesson-01-blockchain-basics/README.md)** - 理解哈希、区块和私钥。
- **[第02课：第一个智能合约](lesson-02-first-smart-contract/README.md)** - 编写、编译并部署你的存钱罐合约。
- **[第03课：Go 语言连接合约](lesson-03-go-connect-contract/README.md)** - 学习如何使用 Go 语言与链上合约交互。
- **[第04课：代币转账](lesson-04-transfer-balance/README.md)** - 实现并读取代币余额与转账。

### 第二阶段：进阶实战 (Intermediate)
- **[第05课：去中心化投票系统](lesson-05-voting-system/README.md)** - 编写一个公平、透明的投票合约。
- **[第06课：发行你的代币 (ERC20)](lesson-06-simple-token/README.md)** - 遵循 ERC20 标准发行自己的加密货币。
- **[第07课：NFT 开发实战 (ERC721)](lesson-07-nft/README.md)** - 铸造独一无二的数字艺术品。
- **[第08课：去中心化交易所 (DEX)](lesson-08-dex/README.md)** - 理解 AMM 机制并实现代币交换。
- **[第09课：DeFi 进阶](lesson-09-advanced-defi/README.md)** - 质押挖矿与流动性挖矿原理。
- **[第10课：合约安全](lesson-10-security/README.md)** - 常见的黑客攻击手法（重入攻击等）与防御。

### 第三阶段：高级架构 (Advanced)
- **[第11课：可升级合约](lesson-11-upgradeable/README.md)** - 代理模式与合约升级实战。
- **[第12课：预言机 (Oracle)](lesson-12-oracle/README.md)** - 使用 Chainlink 获取链下价格数据。
- **[第13课：闪电贷 (Flash Loan)](lesson-13-flashloan/README.md)** - Aave 闪电贷套利原理与实现。
- **[第14课：DAO 治理](lesson-14-dao/README.md)** - 搭建完整的去中心化自治组织（治理代币+时间锁）。
- **[第15课：Layer 2 扩容](lesson-15-layer2/README.md)** - 在 Optimism/Arbitrum 上部署合约。

## 🛠️ 环境准备

在开始之前，请确保你已经安装了以下工具：

1.  **Node.js** (v16+)
2.  **Go** (v1.19+)
3.  **Metamask** 钱包插件

## 🚀 快速开始

克隆本仓库到本地：

```bash
git clone https://github.com/zfchen163/learnsolidity.git
cd learnsolidity
```

进入任意课程目录（例如第13课），运行脚本即可体验：

```bash
cd lesson-13-flashloan
./run.sh
```
